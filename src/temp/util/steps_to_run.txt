1) make .c file which runs stuff
2) make java file with native method
3) complie java file from the top of the package directory
4) complie c code:
 gcc -I/usr/lib/jvm/java-7-openjdk-amd64/include -shared -o HelloJNI.so -fPIC HelloJNI.c

example using HelloJNI:

javac HelloJNI.java
javah HelloJNI
gcc -I/usr/lib/jvm/java-7-openjdk-amd64/include -shared -o libhello.so -fPIC HelloJNI.c
